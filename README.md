# Sloaner 文本处理工具

一个简单易用的在线文本格式转换工具，支持JSON、YAML、XML和Dict格式之间的相互转换。

## 主要功能

- 支持多种格式转换
- 实时数据预览功能
- 一键复制转换结果
- 支持文件下载
- 文件上传功能
- 数据结构可视化

## 技术栈

- HTML5
- TailwindCSS
- Vanilla JavaScript
- RemixIcon

## 使用方法

1. 在输入区域粘贴或输入要转换的数据
2. 选择输入数据的格式（JSON/YAML/Dict）
3. 选择期望输出的格式
4. 点击转换按钮进行转换
5. 使用预览功能查看数据结构
6. 复制或下载转换结果

## 本地开发

1. 克隆项目到本地
2. 点击index.html文件即可开始使用
